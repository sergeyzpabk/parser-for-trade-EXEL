Пример возможности Google Sheet для сбора данных сайтов внутри самой таблицы, без использование сторонных методов. 
Так как есть ограничение работы в 6 минут, необходимо повторно нажать на кнопку сбора данных. 

Проект старый. Для ускорение сбора данных, необходимо переписать логику. А именно сбор данных должен происходить в массив данных(оперативная память), а после уже отрисовываться в самой таблицы, это ускоряет работу парсера.
